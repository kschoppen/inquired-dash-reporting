# Repo memory for Claude

## Git workflow — no pull requests

Kelsey does not use pull requests. At the end of every session, commit the work on
the session's working branch, then merge that branch straight into `main` and push
`main`. Kelsey has given standing permission to push to `main` — do not open a PR,
and do not ask first.

**Exception — stop and ask before merging** if the merge into `main` hits
conflicts. Summarize the change and the conflict and wait for Kelsey's go-ahead
instead of resolving and merging automatically.
